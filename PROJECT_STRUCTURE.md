## File Descriptions

### Core Application Files

**agent.py** (17,387 bytes)
- Multi-agent system implementation
- 4 specialized agents: Classifier, Researcher, Writer, Validator
- CustomerSupportOrchestrator for workflow coordination
- Integration with Google Gemini AI
- Error handling and fallback responses

**tools.py** (7,507 bytes)
- `FAQSearchTool`: Searches FAQ database with keyword matching
- `EmailResponseTool`: Logs/sends customer responses
- Helper functions: `search_faq()`, `send_response()`
- Relevance scoring algorithm

**api_server.py** (5,772 bytes)
- FastAPI REST API server
- Endpoints:
  - POST `/api/support/inquiry` - Submit customer questions
  - GET `/api/support/health` - Health check
  - GET `/api/support/stats` - Usage statistics
- CORS middleware for web demo
- Auto-generated API documentation at `/docs`

**faqs.json** (4,631 bytes)
- Structured FAQ database
- 4 categories: account, billing, technical, general
- 12 question-answer pairs
- JSON format for easy parsing

## Key Directories

### .venv/ (Virtual Environment)
Auto-generated Python virtual environment containing:
- Python 3.12+ interpreter
- All installed dependencies from requirements.txt
- Isolated from system Python

**Size**: ~50MB (varies with dependencies)

**Key subdirectories**:
- `Lib/site-packages/` - Installed Python packages
- `Scripts/` - Python executables (python.exe, pip.exe)
- `Include/` - C headers for extensions

### __pycache__/ (Python Cache)
Auto-generated bytecode cache for faster module loading
- Contains `.pyc` files
- Safe to delete (regenerates automatically)
- Improves startup performance
